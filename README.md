# HCMC - STUDENTS

## Requirements
* Node js
* Express js
* [Mongoose](http://mongoosejs.com/docs/)

## Run Server

Install module

```
npm install
```

Run server by nodemon
```
npm run start
```
Using postman to test api. API run with port 3000

Run web admin
```
npm run admin
```
Using browser run web admin with port 9090
